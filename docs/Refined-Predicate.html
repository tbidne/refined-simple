class Predicate p a where Source #

This class is used for defining new predicates.

We include instances for type-level lists so that we can prove multiple predicates simultaneously.

Since: 0.1.0.0

satisfies :: Proxy p -> a -> Maybe RefineException Source #

Validates predicate p for the type a. Returns Nothing on success, Just RefineException on a validation failure.

data Not p Source #

Logical negation.

Examples

Nothing

>>> satisfies (Proxy @(Not Positive)) 7
Just (MkRefineException {predRep = Not (GreaterThan 0), targetRep = Integer, msg = "7 does not satisfy (Not (GreaterThan 0))"})
